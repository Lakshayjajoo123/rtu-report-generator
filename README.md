# RTU Report Generator

A web-based report generation system that automatically creates RTU-compliant project reports.

## Features

* FastAPI Backend
* DOCX Report Generation
* RTU Cover Page
* Certificate Page
* Declaration Page
* Acknowledgement Page
* Automatic Chapter Generation
* User-configurable Formatting

## Tech Stack

* Python
* FastAPI
* Pydantic
* python-docx
